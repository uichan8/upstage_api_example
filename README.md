# Upstage API Example

Upstage AI API를 사용한 예제 프로젝트입니다.

## 설치

```bash
pip install -r requirements.txt
```

## 환경변수 설정

1. `.env.example` 파일을 `.env`로 복사:
```bash
cp .env.example .env
```

2. `.env` 파일을 열어서 실제 API 키로 수정:
```
UPSTAGE_API_KEY=your_actual_api_key_here
```
